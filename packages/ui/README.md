# @oop-journey/ui

UI component library for the Python OOP Journey application.

Built with:
- Tailwind CSS
- Radix UI primitives
- Lucide icons

## Usage

```typescript
import { Button, Card, Badge, Tabs, Dialog, Sidebar } from '@oop-journey/ui';
```

## Components

### Base Components
- **Button** - Action buttons with variants
- **Card** - Content containers
- **Badge** - Status indicators
- **Separator** - Visual dividers
- **ScrollArea** - Custom scrollbars
- **Tabs** - Tabbed interfaces
- **Accordion** - Collapsible sections
- **Dialog** - Modal dialogs
- **Tooltip** - Hover information
- **Skeleton** - Loading states

### Navigation Components
- **Sidebar** - Collapsible navigation
- **NavLink** - Navigation links
- **Breadcrumbs** - Page hierarchy
- **MobileNav** - Mobile drawer

## Styling

Import the global styles in your layout:

```typescript
import '@oop-journey/ui';
```

Or import the CSS directly:

```typescript
import '@oop-journey/ui/dist/styles/globals.css';
```
