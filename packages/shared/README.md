# @oop-journey/shared

Shared types and utilities for the Python OOP Journey application.

## Usage

```typescript
import type { Curriculum, Week, Day, Problem, User, Progress } from '@oop-journey/shared';
```

## Types

- **Curriculum** - Root curriculum structure
- **Week** - Week metadata and content
- **Day** - Daily learning content
- **Problem** - Coding problems and exercises
- **Project** - Mini and capstone projects
- **User** - User profile
- **Progress** - Learning progress tracking
- **Draft** - Saved code drafts
- **Bookmark** - Bookmarked content
- **ExecutionResult** - Code execution output
- **VerificationResult** - Test verification results
